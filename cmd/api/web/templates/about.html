About Shorty

Shorty is an open-source, production-grade URL shortener built on AWS serverless primitives: Lambda (Go, ARM64), API Gateway v2, DynamoDB, ElastiCache Redis, SQS FIFO, and CloudFront with WAF.

Why another shortener?

Because every SaaS shortener is either slow, expensive, ad-riddled, or all three. We wanted the kind of service we would pay for — but runnable on your own infrastructure for the cost of a coffee per month.

Design goals

  • p99 redirect latency < 100 ms — Redis-first lookup, CloudFront at the edge.
  • Async analytics — click events never block the redirect.
  • Spec-first API — OpenAPI 3.0 drives the Go stubs and BDD tests.
  • Infrastructure as code — the whole stack is Terraform.
  • Testable locally — LocalStack + docker-compose reproduce production faithfully.

Source & contributing

The project lives at github.com/romanitalian/shorty. Issues, RFCs, and PRs welcome.
